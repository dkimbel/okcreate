okCycle

Tiny, modular, flexible slideshow

Demonstration

Fading Transition with Autoplay UI

Stop the autoplay by hovering the slideshow. This example (and others) also show the 'caption', 'navigation'(prev/next buttons), 'pagination' and 'currentPage' UI. 

          
$("#example-1").okCycle({
  autoplay: true,
  effect:'fade', 
  ui: ['caption','navigation','pagination','currentPage']
});

Scrolling Transition with Caption UI

You can also use your mousewheel on this example

            
$("#example-2").okCycle({
  effect:'scroll',
  ui: [
    'caption',
    'navigation',
    'pagination',
    'currentPage', 
    'mouseWheel'
  ]
});

The normal behavior is to use the elements data-caption attribute to provide captions. However, if the data-caption is prefixed with an octothorpe (#), it will be treated as a selector and the innerHTML for the corresponding element will be used.

Sliding Transition with Deferred Loading

Image loading can be deferred until a given item becomes visible by using this loadOnShow option. Important - you must use a data property to set the image source to prevent the browser from automatically downloading the image. This is defined by the dataAttribute option and be default is src. If you forget to do this, nothing bad will happen, you simply won't get the benefit of deferred loading. 

          
// Where 'W' and 'H' are the width and height of your image
<img src='placeholder.png' data-src='/path/to/image' alt='' width='W' height='H' />

Additionally we are making use of image loading callbacks, onProgress and onDone, in order to show loading progress 

          
$("#example-3").okCycle({
  effect:'slide', 
  loadOnShow: true, 
  ui: ['caption','navigation','pagination','currentPage'],
  onProgress: function(self, data, image){
    var width = (data.loaded / data.total) * 100;

    $(image).fadeIn();

    $("#progress > div").css({width: width + '%'});
  },
  onDone: function(self,options){
    setTimeout(function(){
      $("#progress > div").css({
        backgroundColor: '#fff'
      }).html("All loaded");
    },500);
  }
});
