okStylize

Styling for unstyleable form elements: checkboxes, radios, selects and file inputs

Note that as usual the styles are NOT part of the okcreate! framework, but are simply the demo styles.

okStylize can be used to style ANY input - though you probably shouldn't, because it is unnecessary!

          
$("form").okStylize();
